# 🍳 Ne Pişirelim?

Aile için akıllı yemek planlama uygulaması.

## Özellikler

- 🤖 AI ile günlük 3 menü önerisi (Groq)
- 🗳️ Aile içi oylama sistemi
- 📦 Dolap malzeme yönetimi
- 📷 Fotoğraf ile malzeme tanıma
- 🛒 Alışveriş listesi
- 👤 Kişisel profil ve tercihler
- ⏰ Otomatik günlük menü oluşturma

## Kurulum

### 1. Gereksinimler
- Python 3.11+
- Firebase hesabı
- Groq API key

### 2. Kurulum

```bash
git clone https://github.com/kullaniciadin/ne-pisirelim.git
cd ne-pisirelim
